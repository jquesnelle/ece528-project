rugmi

rugmi is a simple image host and converter written by Jeffrey Quesnelle as a final project for ECE 528 (Cloud Computing) at University of Michigan-Dearborn.

rugmi's front end uses the Bootstrap framework and Dropzone.js for file uploading. The back end is written in Python using the Flask microframework. The app is hosted on Google App Engine. Metadata on uploads is stored in Google Cloud Datastore (a NoSQL database) and files themselves are served using Google Cloud Storage.
